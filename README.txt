NHÓM 4 - HAI CHÚ ĐÓM CON

Một số lưu ý trong chương trình:
    1. Check đường dẫn gọi Pytesseract vì mỗi thiết bị có thể lưu đường dẫn ở vị trí khác nhau trên ổ đĩa.
    2. Lưu ý khi mở file, mở đúng folder Nhom4-DuongNghi, không mở toàn bộ folder chứa cả file pdf báo cáo kèm theo vì 
    khi đó sẽ dẫn đến việc sai đường dẫn gọi sample. Cụ thể để chạy được, đường dẫn phải là samples/ten_sample.jpg. Nếu mở folder 
    ngoài kèm theo có file pdf báo cáo, đường dẫn sẽ bị đổi thành Nhom4-DuongNghi/samples/ten_sample.jpg => không chạy được ạ :<
    3. Trong file có môi trường ảo đã có kèm theo file requirements.txt, file này bao gồm tất cả các thư viện sử dụng trong chương trình.
    Để cài đặt, chuyển đổi qua môi trường ảo, nhập câu lệnh sau đây dưới terminal: pip install -r requirements.txt
    4. Các file PDF trong VS bị mã hóa hoặc không xem được, để có thể xem file, chọn file cần xem -> chuột phải -> Reveal in File Explorer
    và double click vào file cần xem.
    5. Folder outputs là các folder chứa ảnh sau khi đã scan.

Đây là một đồ án khá là khó khăn, nhóm em đã cố gắng hoàn thành một cách chỉnh chu nhất có thể nhưng vẫn khó tránh khỏi những sai sót
và hạn chế vẫn còn tồn tại trong đồ án. Nhóm em rất mong thầy sẽ thông cảm vì các bước hạn chế mà tụi em chưa có thể linh động được. 
Tụi em cảm ơn thầy rất nhiều ạ!
